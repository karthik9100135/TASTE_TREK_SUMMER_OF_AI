## How to Contribute

1. **Fork the repository**
2. **Create a new branch** for your feature or bugfix
3. **Commit your changes**
4. **Push to your forked branch**
5. **Create a Pull Request**

---

## Guidelines

- Stick to modular code (each function or component should be independent).
- Ensure Streamlit UI changes don't break existing pages.
- Add comments and docstrings wherever necessary.
- Test your changes locally before pushing.

---

## Areas You Can Contribute To

- New recipe APIs or dataset integrations
- UI enhancements (image gallery, ingredient suggestions, etc.)
- Speech-to-text improvements using other models
- Adding multilingual support
- Writing tests
